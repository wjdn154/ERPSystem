## 1. 프로젝트 개요 (Project Overview)

이 프로젝트는 클라우드 환경에서 확장 가능한 ERP 시스템을 구축하기 위해 세 단계로 진행되었습니다.  
각 단계는 기술적 발전을 바탕으로 점진적으로 확장되었으며, 최종적으로 완성된 ERP 시스템 구축을 목표로 했습니다.
